# Extractor-For-Renpy-s-.rpa-Archive
A simple extractor for RenPY's .rpa archive. 
简单的renpy rpa文件解压工具.

用法Usage:

>python rpa.py c:\\\\example.rpa

或者Or:

>python rpa.py
>\>\>\>c:\\\\example.rpa

非常非常非常简单的rpa解压工具
不依赖第三方库(包括renpy)
python版本要求在3.5及以上
支持windows
支持最新版本renpy
不支持加密文件
会在rpa文件所在的文件夹创建一个与文件同名的文件夹
